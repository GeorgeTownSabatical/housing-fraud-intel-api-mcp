# Sample Data

## Synthetic

All records under `sample_data/synthetic/` are fictional. They exist to exercise lawful investigative, provenance, graph, and risk-scoring workflows without using real people or real properties.

## Real Public Sample

`sample_data/real_public_sample/` is reserved for small lawful public-record samples only when redistribution is clearly permitted by the source. If redistribution is unclear, keep only schemas, scripts, and manifests in the repository and store raw files locally under `data/raw/`.
